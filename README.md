## Pour qui ?

- Étudiant·es en géographie, aménagement, urbanisme (L3, M1, M2)
- Profils en reconversion design / graphisme → SIG
- Enseignant·es cherchant un fil conducteur pour structurer un cours QGIS
- Autodidactes souhaitant un parcours guidé sur 8 semaines

## Ancrage territorial

Tous les exercices utilisent des données réelles centrées sur **Toulouse et la Haute-Garonne (31)** pour offrir une progression cohérente et reproductible.

## Sources de données utilisées

- **IGN** : Admin Express COG, OCS GE, RGE ALTI 5m, Contours IRIS
- **INSEE** : Recensement de la population, Base Permanente des Équipements
- **OpenStreetMap** : via le plugin QGIS QuickOSM
- **Géoportail Urbanisme** : PLU, PPRI (bonus)

Toutes ces données sont gratuites et librement téléchargeables sur les portails officiels (`geoservices.ign.fr`, `insee.fr`).

## Pré-requis

- QGIS LTR (3.34 ou ultérieur) installé — [téléchargement officiel](https://qgis.org/fr/site/forusers/download.html)
- Aucune connaissance SIG préalable nécessaire
- Aucun pré-requis en programmation
